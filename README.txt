== Description ==

WP Adf.ly Dashboard and Integration is a plugin allows you to configure Full Page Script, Website Entry Script, Pop-Ads tools and Dashboard widget for stats

== Installation ==

1. Upload `wp-adf-ly-dashboard-integration` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the plugin settings by clicking "AdFly Integration Settings" on the settings navbar

= Where can I get the user id, public api key and secret api key? =
